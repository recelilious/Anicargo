# Anicargo

Anicargo is a private-deployment anime subscription, download, and playback project for small trusted groups.

## Planned Stack

- Backend: Rust
- Web frontend: React + TypeScript + Fluent 2
- Metadata: Bangumi API
- Resource discovery: AnimeGarden API
- Native client: WinUI 3 (placeholder only for now)

## Initial Layout

- `backend/`: Rust backend workspace placeholder
- `frontend/web/`: React web client placeholder
- `clients/winui3/`: future WinUI 3 client placeholder
- `shared/`: shared contracts, schemas, and tooling
- `deploy/`: deployment and compose-related files
- `scripts/`: local helper scripts
- `ignore/docs/`: initialization notes and internal reports

## Current Status

This commit only initializes the repository structure, `.gitignore`, and the first feasibility report in `ignore/docs/init_report.md`.
